# Product Specification

## Summary

AI Tools History Archive is a source-linked public archive for tracking AI tools, services, products, APIs, models, features, and hardware over time.

It focuses on lifecycle events such as shutdowns, acquisitions, rebrands, deprecations, retirements, feature removals, pricing changes, and major product transitions.

## Purpose

The project exists to make AI product changes easier to verify and understand. It should answer public questions such as:

- What was this AI tool or product?
- Who operated it?
- What changed?
- When did the change happen?
- Is it still available?
- Was it rebranded, merged, deprecated, retired, or shut down?
- What public evidence supports the record?

## Scope

In scope:

- AI tools and web services
- AI products and product families
- AI APIs and model offerings
- AI features within larger products
- AI hardware and devices
- Public lifecycle events with source-backed evidence

Out of scope for the initial version:

- AI tool rankings
- User reviews
- Recommendation lists
- Automated scoring
- Unsourced rumors
- Private or leaked information
- General AI safety incident tracking unless directly tied to a product lifecycle event

## Core Pages

Planned public pages:

- `/` home
- `/records/` all records
- `/records/[slug]/` record detail
- `/events/` event index
- `/shutdowns/` shutdown-related events
- `/acquisitions/` acquisition and merger events
- `/rebrands/` rebrand and transition events
- `/api-model-deprecations/` API and model lifecycle events
- `/categories/` category index
- `/stats/` archive coverage and composition
- `/methodology/` definitions and rules
- `/glossary/` terms
- `/submit/` correction and submission path
- `/about/` project explanation

## v0 Acceptance Criteria

The first public version should include:

- White-background neutral archive UI
- Static-friendly implementation
- Public records list
- Record detail pages
- Timeline and evidence display
- Status and confidence labels
- API/model deprecation index
- Category index
- Stats page
- Methodology and evidence policy
- Correction/reporting path
- Initial reviewed seed records

## Non-Goals

This project should not become:

- A best AI tools directory
- A popularity ranking
- A hype dashboard
- A graveyard joke site
- A private strategy repository
- A public log of internal workflow decisions
